Предназначение
Передача и хранение файлов в облачном пространстве.
Функционал включает в себя подробный просмотр отдельно взятых или всех загруженных файлов и информации о них через поиск, возможность делиться ими с помощью ссылок и комментировать.
Технологии
Помимо основных (CSS3, HTML5), используются jQuery 3.3.1, AJAX, PHP 7.2 и Apache.
Поиск
Поисковая строка обладает переключателем, отвечающим за восприятие запроса системой.
Название - поиск по имени файла, включая расширение.
Хэш MD5 - поиск по контрольной сумме MD5.
Послав запрос с пустым полем, выведется последняя часть ленты загруженных файлов. Прокрутка вниз будет постепенно выводить все более старые загрузки.
Регистрация
Открывает доступ к добавлению информации на ресурс.
Уведомления
Файлы
На главной странице, под строкой поиска и на всех страницах, в правом нижнем углу, есть кнопки "Загрузить".
После нажатия на любую из них появится окно с рамкой.
Комментарии
1. Файлы
1.1. Размер одного файла не может превышать 52,428,800 байт.
1.2. Название и описание должно отражать реальное содержание.
1.3. Материал не предназначенный для лиц младше 18 лет требуется помечать соответствующим переключателем.
2. Комментарии
2.1. Спам (реклама, ссылки на сомнительные сторонние ресурсы) и флуд (многочисленные сообщения, не несущие смысловой нагрузки) запрещены.
3. Ответственность
3.1. Файлы или комментарии с обнаруженными нарушениями будут отредактированы либо удалены без предупреждения.
Администрация сервиса оставляет за собой право в любой момент изменять вышеперечисленные пункты.
